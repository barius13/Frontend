# Kythi Frontend


### Install yarn
```
$ npm install --global yarn
```
### Dependencies
```
$ yarn install
$ yarn add global
$ yarn build
```
### Development
```
$ yarn dev
```
### Requirements
* NodeJS
* npm (Node package manager)
* Yarn
